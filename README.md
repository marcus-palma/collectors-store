**collectors-store** is a showcase e-commerce web application with the aim to establish solid framework for future commercial applications within the modern industry

**Current state of development:**
- The Frontend client is under development, before the Backend server is initiated
- **Product Grid:** A modular, multi-use, Single Page Application, Responsive Web Design component for getting and displaying data of **Product Entry** (see description in ``client/js/product_grid.js`` for details). The Product Grid is nearly ready for implementation. Once implemented, the Backend server will be initiated and developed to serve Product Entry content dynamically.
- **Header:** A modular, Responsive Web Design component with navigation controls is ready to be implemented into the top of the e-commerce application.

**Current Tech stack for Frontend:**
- JavaScript
- HTML
- CSS
- Web API

**Planned Tech stack for Backend:**
- Node.js for server
- PostgreSQL for database
